# 中国监管动态来源目录

用于 `reg-feed-watcher` 和监管差距分析类 skill。检索时优先使用官方来源，并记录检索日期、发布日期、施行日期、征求意见截止日和是否已有正式稿。

| 领域 | 来源 |
|---|---|
| 法律/立法 | 全国人大、中国人大网、国家法律法规数据库 |
| 行政法规/政策 | 中国政府网、国务院公报 |
| 司法解释/案例 | 最高人民法院、最高人民检察院、人民法院案例库 |
| 网信/数据/AI | 中央网信办、国家数据局、网安标委 |
| 市场监管/广告/反垄断 | 国家市场监督管理总局、反垄断执法机构、地方市场监管局 |
| 工信/互联网/电信 | 工业和信息化部、通信管理局 |
| 金融 | 中国人民银行、国家金融监督管理总局、中国证监会、交易所 |
| 医药/医疗器械 | 国家药监局、国家卫健委 |
| 劳动用工 | 人社部、地方人社部门、社保/公积金中心 |
| 知识产权 | 国家知识产权局、版权局、法院知识产权庭 |
| 商务/外资/出口管制 | 商务部、发改委、海关总署、外汇局 |
| 标准 | 全国标准信息公共服务平台、网安标委、行业标准发布平台 |

对征求意见稿，输出中使用“公开征求意见/意见反馈截止日”，不要沿用 NPRM 术语。
